# Cola Music · 可乐音乐

A native Android client for your self-hosted music server. Built for Chinese-language libraries, lossless-first playback, and large (20,000+ song) collections. Supports Navidrome and other OpenSubsonic-compatible servers.

- **Download:** <https://github.com/redblack168/cola-music/releases/latest>
- **Source:** <https://github.com/redblack168/cola-music>
- **Privacy policy:** [PRIVACY](PRIVACY)

## Highlights

- Media3 / ExoPlayer pipeline with FLAC + ALAC original-stream playback.
- Chinese text normalization (S↔T fold, full-width → half-width, noise-token stripping) for search and lyric matching.
- Four-source lyric chain: Navidrome, LRCLIB, optional NetEase, optional QQ Music. Scored, cached, and manually rematchable.
- Offline downloads via Media3 DownloadManager, Wi-Fi gate by default.
- Foldable-aware Compose UI (WindowSizeClass-driven layout).
- No accounts, no analytics, no telemetry. Everything stays between your phone and your server.
